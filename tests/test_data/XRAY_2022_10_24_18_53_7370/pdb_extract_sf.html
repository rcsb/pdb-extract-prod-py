This table is used to multiple diffraction data sets into one mmCIF file with details.

You can use this table if you want to merge all the SF files (which were obtained seperately from different sources) in one mmCIF format.

  • Crystal ID : the identification of the crystal used for diffraction.
  • Wavelength value : the wavelength of Xray used for diffraction.
  • Diffrac. details : give details how you obtain the data set (e.g beamline, temperature, methods...).
  • Data file name : the name of file containing structure factors.

    After you run the program, all the data sets will be merged into one mmCIF file for which each data set is identified by the IDs.
